### Cove Plugin Typing Indicator 修复 (22:00-23:10)
- **根因**: PR #222 移除服务端 opcode 4，但 plugin 编译产物还在用 op:4 发 typing
- **踩的坑**: 用 tsc 重新编译 plugin（40 个 type error），产出散文件和原版 esbuild bundle 完全不同，导致 fetch failed、消息完全不通
- **正确做法**: 
  1. 源码改 op:4 → restClient.sendTyping()
  2. pnpm install 安装 ws 依赖
  3. 用原版同样的 esbuild 命令打包: `npx esbuild src/index.ts --bundle --platform=node --format=esm --outfile=dist/index.js --external:openclaw --alias:@cove/shared=../shared/src/index.ts`
  4. 复制到 extensions/cove/dist/index.js → 重启 gateway
- **教训**: 改 plugin 必须用正确的 build 工具链（esbuild bundle），不能用 tsc 替代。改之前先搞清楚原版怎么打包的。
