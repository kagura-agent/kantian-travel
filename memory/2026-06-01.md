**Applied**: 修复 `scripts/weekly-eval.sh` line 72 integer comparison error

**Commit**: 975da60 pushed to kagura-agent/dna
**Lesson**: `grep -c || echo 0` 是 bash 经典陷阱 — grep -c 总输出到 stdout（即使 0），exit code 才表示有无匹配。用 `|| true` 更安全
